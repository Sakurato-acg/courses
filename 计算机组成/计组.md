>
> 2. **CPI**：执行一条指令所需的时钟周期数
>

### 性能分析

设tc为访问一次Cache所需时间，tm为访问一次主存所需的时间。

`命中率H`：CPU欲访问的信息已在Cache中的比率

`缺失率M`：M = 1 - H

`平均访问时间`：t = H * tc + ( 1 - H ) * ( tc + tm ) 

​							或  t = H * tc + ( 1 - H ) * tm

### Cache与主存的映射

> 分类

1. `全相联映射`：主存块可以放在Cache的任意位置

2. `直接映射`：每个主存块只能放到一个特定的位置Cache块号 = 主存块号 % Cache总块数
